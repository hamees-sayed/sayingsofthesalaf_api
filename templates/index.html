# Method Endpoint Description
1. GET /narrations All the narrations.
2. GET /narration/`int:id` Narrations by a specific id.
3. GET /names Names of the scholars.
4. GET /name/`string:name` All the narrations by a specific scholar.
5. GET /topics All the topics.
6. GET /topic/`string:topic` All the narrations by a specific topic.

Example:

  1. https://sayingsofthesalaf-api.onrender.com/narrations
  2. https://sayingsofthesalaf-api.onrender.com/narration/5
  3. https://sayingsofthesalaf-api.onrender.com/names
  4. https://sayingsofthesalaf-api.onrender.com/name/abul-jawza
  5. https://sayingsofthesalaf-api.onrender.com/topics
  6. https://sayingsofthesalaf-api.onrender.com/topic/advice

Parameters:

- int: id [1, 499] (Range for the ids of the narrations)
